This website provides the Haiku REST api. The following commands are available: 
       
            GET /haikus                   Get all haikus
            GET /haikus/random            Get a random haiku
            POST /haikus                  Create a new haiku

            GET /haikus/:id               Get a single haiku by id
            DELETE /haikus/:id            Delete a haiku

            POST /haikus/:id/like         "Like" a single haiku

            GET /author/:name             Get all haikus authored by :name
